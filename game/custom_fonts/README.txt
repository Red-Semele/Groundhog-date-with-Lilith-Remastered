========================================
  CUSTOM FONTS FOLDER
========================================

Want to use your own font in the game? Drop your font file in this folder!

Supported formats:  .ttf  and  .otf

HOW TO USE
----------
1. Copy your font file (e.g. OpenDyslexic-Regular.ttf) into this folder.
2. Fully close and restart the game.
3. Open Preferences → Font.
4. Your font will appear at the bottom of the list under "Custom", labelled
   by its filename (hyphens and underscores are replaced with spaces).
5. Click it to switch. The change is saved and will be applied every time
   you start the game.

TIPS
----
- Dyslexia-friendly fonts like OpenDyslexic or Lexie Readable work great here.
- Any font that is free for personal use is fine to add.
- You can add as many fonts as you like; they will all be listed.
- To remove a font, delete its file and restart the game.
  (If you had that font selected, the game will fall back to DejaVu Sans.)

FINDING FREE FONTS
------------------
- https://fonts.google.com           (search "dyslexia", "readable", etc.)
- https://opendyslexic.org           (OpenDyslexic - free dyslexia font)
- https://www.fontsquirrel.com
